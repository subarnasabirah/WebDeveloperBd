someObject._name = 'Subarna'; -> _name keyword is a valid keyword and it is possible to assign data.


someObject.age = 6; -> age keyword is a valid keyword and it is possible to assign data.


someObject[prop] ='red'; -> [prop] keyword is a valid process and thus data can be updated.


someObject.123 = true; -> 123 keyword is a invalid keyword and it is not possible to assign data. if we want to assign data, then you have to write like this someObject.['123 ']= true;
